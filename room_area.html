PROJECT 2: ROOM AREA

Irregular shapes are a common occurrence in the real world. The goal was to use Python to create a calculator that functions as an area finder for shapes such as these. To demonstrate the knowledge of Python as well as how it applies to the real world, create a simple program that can be used to find room area using mathematical expressions and variables.

Inspiration behind this stemmed from the translation of real world problems into something solvable by using Python.
