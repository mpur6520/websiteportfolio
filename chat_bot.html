PROJECT 3: CHATBOT

Using Python, the end goal was to create a program where the user can interact with and answer questions given, and the computer mimics a human’s response at random—the final result should contain a program that runs questions and responses simulating a person the user can have a full conversation with.

The inspiration came from human conversations, I wanted to mimic this through the program despite the responses being randomized. I’m especially proud of how well the randomization worked because it felt more authentic, but it was an overall struggle to think of different questions for the program to ask the user. I’ll most likely refer back to this whenever I need to know more about how to create message simulations.
