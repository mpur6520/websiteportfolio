BUILD A BEAT

Participate in a competition regarding music production in Earsketch, using Python to create a musical piece. The final product should be a creative song where coding techniques are used to optimize efficiency, where skills learned with Python are put to use.

This was my second time coding music in Earsketch and my inspiration this time was the guitar. I attempted to make a melody based around it utilizing my knowledge of Earsketch from the first time—the hardest part was choosing what sounds to use, but I’m especially proud of how much easier the functions I used were this time around. It was a pleasant surprise to find that none of the hits overlapped this time.
