PROJECT 1: SILLY SENTENCES

Using Python, create a paragraph the user can alter using the input feature and variables; this way, Python can become familiar and knowledge of how elements interact with each other could be shown. The end product includes a paragraph detailing a trip, changed with the words they’ve inputted.

The project was easy as I was given a guideline on what it should look like, the most difficult thing was getting the syntax right as the grading was very particular. The rest was easy, and I’m proud of how much I could accomplish considering that at this point, I hadn’t been learning Python for long.
